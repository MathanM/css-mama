LENGTH UNITS

Length is a number followed by a length unit. A whitespace cannot appear between the number and the unit.

If the value is 0, the unit can be omitted. The length units are of two types.

Absolute Length Relative Length

Absolute Length

The length units which are fixed and which doesn't vary according to the screen resolution are called Absoulte lengths.

cm centimeters
mm millimeters
in inches
px pixels
pt points
pc picas

Relative Length

The length units which are not fixed and relative to the another length property like font-size, width and height of parent or viewport are called Relative lengths.

em Relative to the font size of the element
ex Relative to the x-height of the current font
ch Relative to width of the "0"
rem Relative to font-size of the root element
vw Relative to 1% of the viewport width
vh Relative to 1% of the viewport height
vmin Relative to 1% of the viewport smaller dimension (smaller of 1vh or 1vw)
vmax Relative to 1% of the viewport larger dimension (greater of 1vh or 1vw)
% Relative to the parent element
